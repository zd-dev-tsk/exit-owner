# Exit Owner

Free founder dilution simulator. Stack SAFEs, priced rounds, option pool top-ups, and 1x liquidation preferences — and see exactly what each founder takes home at exit. No signup, no backend; everything runs in your browser.

**Live site:** https://zd-dev-tsk.github.io/exit-owner/

## Run locally

Open `index.html` in a browser, or serve the folder:

```sh
python3 -m http.server
```

## Tests

```sh
node test.js
```
